### Square root

## Algebra

### Solving for x
ax + b = c
x = (c - b) / a

### Quadratic formula
Given `ax^2 + bx + c = 0`, then `x = (-b +- sqrt(b^2-4ac)) / 2a`

### Pythagorean theorem
c = sqrt(a^2 + b^2)
